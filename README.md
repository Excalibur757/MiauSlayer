# MiauSlayer

## ✅ Como rodar o jogo

1. Baixe o Godot (versão X.Y.Z).
2. Abra o `project.godot`.
3. Pressione Play ▶ para executar.

## ✅ Como rodar os testes

godot --headless --path . -s res://test_runner.gd


## ✅ Build automatizado

Nosso pipeline do GitHub Actions exporta automaticamente o build e publica como artefato.

## ✅ Download

Confira os artefatos de build na aba **Actions** → **último workflow** → **Download artifacts**.
